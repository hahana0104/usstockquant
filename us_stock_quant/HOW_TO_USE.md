# 美股三因子量化策略使用说明

## 系统简介

**策略逻辑**: Quality(质量) + Value(价值) + Momentum(动量)
- **质量因子**: ROE>15%、盈利稳定性
- **价值因子**: PE/PB低于行业中位数  
- **动量因子**: 6个月价格趋势强度

**使用方式**: AI分析建议 → 你自主决策 → 手动下单

## 结果解读

### 综合得分 (Score)
- **0.7+**: 强烈推荐
- **0.6-0.7**: 推荐
- **0.5-0.6**: 中性
- **<0.5**: 不推荐

### 盈亏百分比
- **+20%以上**: 考虑止盈
- **-10%以下**: 考虑止损
- **策略建议卖出 + 盈利**: 止盈信号
- **策略建议卖出 + 亏损**: 止损信号

## 风险提示

⚠️ **重要提醒**:
1. **历史回测不代表未来收益**，策略也可能亏损
2. **建议先用模拟盘观察1-3个月**，熟悉策略风格
3. **不要全仓跟随**，建议只用部分资金试水
4. **定期复盘**，如果策略连续跑输大盘，考虑暂停

⚠️ **该策略不适合**:
- 短期追涨杀跌
- 不能承受20%以上回撤
- 希望每天都交易

## 常见问题

**Q: 为什么策略建议卖出我盈利的股票？**
A: 策略基于因子计算，可能认为该股票质量/价值/动量指标已恶化，建议锁定利润。

**Q: 策略选的股票跌了怎么办？**
A: 策略不是100%正确，可能有回撤。建议设置止损线（如-15%），到了就卖。

**Q: 多久跑一次分析？**
A: 策略设计是月度调仓，建议每月1号运行。不必每天看。

**Q: 可以自动交易吗？**
A: 技术上可以，但**强烈建议手动操作**。自动交易有风险，需要你自己承担。
